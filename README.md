# E-commerce-Sales-Analysis-Power-BI-Project
## Project Overview

This project focuses on analyzing e-commerce sales data using Power BI.
The goal is to understand sales performance and customer behavior by visualizing key business metrics in an interactive dashboard.

## Problem Statement

E-commerce companies generate large amounts of sales data, but it is often difficult to track revenue trends and identify top-performing products and regions.
The objective of this project is to create an interactive Power BI dashboard that clearly presents sales insights.

## Objectives

Analyze overall sales performance

Track revenue and order trends over time

Identify top-selling products and categories

Understand customer and regional sales behavior

Support business decision making using visual analytics

## Dataset
The dataset contains e-commerce transaction details such as:
-Order ID

-Customer ID

-Product Category

-Product Name

-Quantity

-Price

-Payment Method

-Order Date

-Region

## Steps Followed

-Loaded the dataset into Power BI

-Cleaned and transformed data using Power Query

-Created calculated columns and measures using DAX

-Built relationships between tables

-Designed interactive dashboards and visuals

-Added filters and slicers for better analysis

## Power BI Features Used

-Power Query for data cleaning

-DAX for calculations and KPIs

-KPI cards for key metrics

-Bar charts, line charts, and tables

-Slicers for interactive filtering

## Key Insights

-Certain product categories generate the highest revenue

-Sales show clear trends over different time periods

-Some regions contribute more to overall sales

-Customer purchasing behavior varies across categories

## Results

-Built an interactive Power BI dashboard for sales analysis

-Improved visibility into sales performance and trends

-Enabled quick and effective business analysis

## Tools & Technologies

-Power BI Desktop

-Power Query

-DAX

-CSV Dataset

## Conclusion

This project helps analyze e-commerce sales data and provides meaningful insights through interactive dashboards.
It can be used by businesses to monitor performance, identify opportunities, and make informed decisions.
