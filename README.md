# E-commerce-Sales-Analysis-Power-BI-Project
Business intelligence project using Power BI to analyze e-commerce sales, identify key trends, and support data-driven decision making.
Project Overview

This project focuses on analyzing e-commerce sales data using Power BI to generate actionable business insights. The objective is to transform raw transactional data into an interactive dashboard that supports decision-making related to sales performance, customer behavior, and product trends.

Problem Statement

E-commerce businesses generate large volumes of sales data but often lack a structured approach to analyze revenue trends, product performance, and customer purchasing behavior. This project addresses that gap by creating a Power BI dashboard that provides clear visibility into key business metrics.

Objectives

Build an interactive Power BI dashboard for sales analysis

Perform data cleaning and transformation

Analyze revenue trends and product category performance

Identify customer purchasing patterns and regional sales distribution

Enable data-driven business decision making

Dataset

The dataset used in this project contains e-commerce transactional records with the following attributes:

Order ID

Customer ID

Product Category

Product Name

Quantity

Price

Payment Method

Order Status

Order Date

Region

Dataset file name: Ecommerce_Sales_Dataset.csv

Steps Followed
Data Cleaning

Removed duplicate records

Handled missing and inconsistent values

Standardized categorical fields

Data Transformation

Created calculated columns such as Revenue (Quantity multiplied by Price)

Extracted Year, Month, and Quarter from Order Date

Built DAX measures for key performance indicators

Data Modeling

Established relationships between fact and dimension tables

Designed a structured data model for analysis

Dashboard Development

KPI cards for revenue, total orders, and average order value

Time-series analysis of sales trends

Product and category performance analysis

Customer and regional segmentation

Interactive filters and slicers

Key Insights

Identified top-performing product categories

Observed seasonal trends in revenue and order volume

Analyzed regions contributing the highest sales

Evaluated customer purchasing behavior

Identified preferred payment methods

Results

The Power BI dashboard provides a centralized view of e-commerce performance, enabling stakeholders to monitor key metrics, identify growth opportunities, and make informed business decisions.

Tools and Technologies

Conclusion

This project demonstrates an end-to-end business intelligence workflow using Power BI, from raw data preparation to interactive dashboard development. It highlights skills in data analysis, visualization, data modeling, and business reporting.
