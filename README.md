# E-commerce-Sales-Analysis-Power-BI-Project
Project Overview

This project focuses on analyzing e-commerce sales data using Power BI.
The goal is to understand sales performance and customer behavior by visualizing key business metrics in an interactive dashboard.

Problem Statement

E-commerce companies generate large amounts of sales data, but it is often difficult to track revenue trends and identify top-performing products and regions.
The objective of this project is to create an interactive Power BI dashboard that clearly presents sales insights.
